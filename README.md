# smart-home-lighting-system
Smart Home Lighting System developed for the CodeAlpha Internship Program. The system uses a PIR motion sensor, LDR sensor, Arduino Uno, and relay module to automate lighting based on motion and ambient light conditions. Designed to improve energy efficiency, reduce power consumption, and enhance user convenience

This project presents the design of a Smart Home Lighting System developed as part of the CodeAlpha Internship Program.

The system automatically controls lighting based on:
- Motion detection using a PIR Sensor
- Ambient light detection using an LDR Sensor

The objective is to improve energy efficiency and user convenience.

## Components Used
- PIR Motion Sensor
- LDR Sensor
- Arduino Uno
- Relay Module
- LED Light
- Power Supply
